## 进程调度与资源管理

### 算法

* 进程调度算法
  * 采用非抢断的算法
  * 使用优先级策略
  * 增了等待时间，在优先级相同时，进程调度有限选择等待时间长的
* 资源管理
  * 对于资源未设定数量
  * 创建进程时也会规定所需资源的数量
  * 在此实验中每个进程规定仅需要一种资源
